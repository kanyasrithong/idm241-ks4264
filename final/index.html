RULES:

Product Card

Hover On

  • Lego Orchid image fades out and Lego Box image fades in over 0.3 seconds

Hover Off

  • LEGO Box image fades out and LEGO Orchid image fades in over 0.3 seconds

Age Indicator

Hover On

  • Birthday cake icon transforms up 4px over 0.3 seconds
  • Cursor changes to finger

Hover Off

  • Birthday cake icon transforms down 4px over 0.3 seconds
  • Cursor changes to pointer

Lego Piece Count

Hover On

  • Lego piece icon rotates 25 degrees over 0.3 seconds
  • Cursor changes to finger

Hover Off

  • Lego piece icon rotates -25 degrees over 0.3 seconds
  • Cursor changes to pointer

Star Rating

Hover On

  • Star icon rotates 144 degrees over 0.4 seconds
  • Star icon turns from black to yellow over 0.1 seconds
  • Cursor changes to finger

Hover Off

  • Star icon rotates -144 degrees over 0.4 seconds
  • Star icon turns from yellow to black over 0.3 seconds
  • Cursor changes to point

Add to Cart Button

Hover On

  • Black background slides in from left to replace orange background over 0.3 seconds
  • Bag icon and "Add to Bag" changes from black to orange over 0.3 seconds
  • Cursor changes to finger

Hover Off

  • Black background color slides out to left to reveal orange background over 0.3 seconds
  • Bag icon and "Add to Bag" text changes from orange to black over 0.3 seconds
  • Cursor changes to finger

Click

  • Checkmark replaces bag icon
  • "Added!" text replaces "Add to Bag" text
  • After delay of 0.8 seconds, black background color slides out to left to reveal orange background over 0.3 seconds
  • After delay of 0.8 seconds, "Add to Bag" text replaces "Added!" text over 0.3 seconds
